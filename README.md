Run-Length Encoding
===================

Author: **[Afaan Bilal](https://afaan.me)**  

## Introduction
**Run-Length Encoding** is a program in C++ implementing the compression algorithm [Run-Length Encoding](https://en.wikipedia.org/wiki/Run-length_encoding).
